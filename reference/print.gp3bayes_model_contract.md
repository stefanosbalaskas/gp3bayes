# Print a gp3bayes Model Contract

Prints a concise summary of an approved model contract. The full object
remains available for programmatic inspection.

## Usage

``` r
# S3 method for class 'gp3bayes_model_contract'
print(x, ...)
```

## Arguments

- x:

  A `gp3bayes_model_contract` object.

- ...:

  Additional arguments. They are currently ignored.

## Value

`x`, invisibly.
