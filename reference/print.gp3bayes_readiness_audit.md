# Print a gp3bayes Readiness Audit

Prints a concise summary of a model-readiness audit and any warnings or
failures. The full structured audit remains available for programmatic
inspection.

## Usage

``` r
# S3 method for class 'gp3bayes_readiness_audit'
print(x, ...)
```

## Arguments

- x:

  A `gp3bayes_readiness_audit` object.

- ...:

  Additional arguments. They are currently ignored.

## Value

`x`, invisibly.
